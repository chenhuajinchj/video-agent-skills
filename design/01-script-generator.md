# 模块 1: video-script-generator

## 功能概述

根据大纲生成带画面标注的逐字稿，是整个流程中最独立的模块。

## 触发条件

- 「生成逐字稿」
- 「根据大纲写稿」
- 「把大纲扩展成完整脚本」

## 目录结构

```
video-script-generator/
├── SKILL.md
└── references/
    ├── script-template.md      # 风格模板
    └── case-sources.md         # 案例引用指南
```

## 输入文件

### outline.md 格式

```markdown
# 视频大纲：[主题]

## 核心主题
[一句话概括]

## 预估信息
- 预估时长：X 分钟
- 预估字数：XXXX 字
- 论点数量：X 个

## 论点结构

### 论点 1：[标题]
- 核心观点：[观点]
- 案例：[案例描述]
- 出处：[来源]

### 论点 2：[标题]
...
```

## 输出文件

### script.md 格式

**重要**：画面标注使用编号格式 `[画面#XXX：描述]`

```markdown
# 逐字稿：[主题]

## 元信息
- 预估时长：X 分钟
- 字数：XXXX 字
- 画面数量：XX 个

---

## 正文

[画面#001：描述画面内容]
第一段文字内容...

[画面#002：描述画面内容]
第二段文字内容...
【出处：来源信息】

...
```

## 风格模板要点

### 开头（三选一）

1. **反常识**：打破常规认知
2. **故事**：从具体场景切入
3. **问题**：抛出引人思考的问题

### 正文要求

- 每个论点必须有案例背书
- 案例必须标注出处：`【出处：来源】`
- 画面标注使用三位数编号：`[画面#001：...]`
- 语言口语化，适合朗读

### 结尾

- 总结核心观点
- 引导关注/互动

## 画面标注规范

| 格式 | 示例 |
|------|------|
| 编号 | `[画面#001：...]` |
| 描述 | 简洁描述画面内容，便于后续素材搜索 |
| 位置 | 放在对应文字段落之前 |

## 检查点

输出逐字稿后暂停，等待用户确认：
- 文案风格是否满意
- 案例是否准确
- 画面描述是否清晰

## 依赖

无外部 Skill 依赖，纯文本生成。

## 使用示例

```
用户：我有一个 outline.md，帮我生成逐字稿
→ 读取 outline.md
→ 生成 script.md
→ 展示预览，等待确认
```
