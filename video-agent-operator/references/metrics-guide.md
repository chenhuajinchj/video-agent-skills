# 运营指标计算指南

## 核心指标公式

### 互动率（Engagement Rate）

```
互动率 = (点赞 + 评论 + 收藏 + 分享) / 播放量 × 100%
```

综合衡量观众参与度的核心指标。互动率高说明内容引发了观众的行动意愿。

### 细分互动指标

```
点赞率 = 点赞 / 播放量 × 100%
评论率 = 评论 / 播放量 × 100%
收藏率 = 收藏 / 播放量 × 100%
分享率 = 分享 / 播放量 × 100%
```

- **点赞率**：反映内容的即时好感度，门槛最低
- **评论率**：反映内容的讨论价值，高评论率说明话题有争议性或共鸣感
- **收藏率**：反映内容的长期价值，知识类内容的核心指标
- **分享率**：反映内容的社交货币属性，高分享率说明观众愿意为内容背书

### 完播率（如有数据）

```
完播率 = 完整观看次数 / 播放量 × 100%
```

抖音算法最看重的指标之一。完播率直接影响推荐权重。

### 粉丝转化率（如有数据）

```
粉丝转化率 = 新增粉丝 / 播放量 × 100%
```

衡量内容对账号增长的贡献。

## 抖音行业基准线

### 知识类账号（growth / ai）

| 指标 | 低迷 | 正常 | 优秀 | 爆款 |
|------|------|------|------|------|
| 互动率 | < 2% | 2-5% | 5-10% | > 10% |
| 点赞率 | < 1% | 1-3% | 3-6% | > 6% |
| 评论率 | < 0.1% | 0.1-0.5% | 0.5-1% | > 1% |
| 收藏率 | < 0.3% | 0.3-1% | 1-3% | > 3% |
| 分享率 | < 0.1% | 0.1-0.5% | 0.5-1% | > 1% |
| 完播率 | < 15% | 15-30% | 30-50% | > 50% |

知识类账号的特点：收藏率和分享率相对较高（内容有实用价值），评论率中等。

### Vlog 类账号（vlog）

| 指标 | 低迷 | 正常 | 优秀 | 爆款 |
|------|------|------|------|------|
| 互动率 | < 3% | 3-7% | 7-15% | > 15% |
| 点赞率 | < 2% | 2-5% | 5-10% | > 10% |
| 评论率 | < 0.2% | 0.2-0.8% | 0.8-2% | > 2% |
| 收藏率 | < 0.1% | 0.1-0.5% | 0.5-1% | > 1% |
| 分享率 | < 0.1% | 0.1-0.3% | 0.3-0.8% | > 0.8% |

Vlog 类账号的特点：点赞率和评论率较高（人格化内容引发情感共鸣），收藏率较低。

## 爆款/低迷判定阈值

### 爆款判定

```
播放量 > 账号近 30 天平均播放量 × 3
```

附加条件（满足任一即可加强判定）：
- 互动率 > 账号平均互动率 × 2
- 24 小时内播放量突破账号历史 TOP 10

### 低迷判定

```
播放量 < 账号近 30 天平均播放量 × 0.3
```

附加条件：
- 互动率 < 账号平均互动率 × 0.5
- 发布 48 小时后播放量仍未突破平均值的 50%

### 注意事项

- 新账号（视频数 < 10）不适用倍数判定，改用行业基准线
- 账号平均值应排除已判定的爆款视频，避免被极端值拉高
- 节假日和热点事件期间的数据需要单独标注

## 时长字段解析规则

抖音导出数据中，时长字段为中文字符串格式，需要转换为秒数：

```python
import re

def parse_duration(duration_str: str) -> int:
    """将 '19分15秒' 格式转换为秒数"""
    total = 0
    hours = re.search(r'(\d+)\s*[时小]时?', duration_str)
    minutes = re.search(r'(\d+)\s*分', duration_str)
    seconds = re.search(r'(\d+)\s*秒', duration_str)
    if hours:
        total += int(hours.group(1)) * 3600
    if minutes:
        total += int(minutes.group(1)) * 60
    if seconds:
        total += int(seconds.group(1))
    return total
```

常见格式示例：
- `"19分15秒"` → 1155 秒
- `"1小时2分30秒"` → 3750 秒
- `"45秒"` → 45 秒
- `"8分"` → 480 秒

## 时长与表现的关系参考

| 时长区间 | 知识类建议 | Vlog 建议 |
|---------|-----------|----------|
| < 1 分钟 | 不推荐（信息密度不足） | 适合日常碎片 |
| 1-3 分钟 | 适合单点知识 | 适合日常记录 |
| 3-8 分钟 | 最佳区间（深度 + 完播率平衡） | 适合主题 Vlog |
| 8-15 分钟 | 适合深度专题 | 不推荐（完播率下降） |
| > 15 分钟 | 仅适合系列课程 | 不推荐 |
