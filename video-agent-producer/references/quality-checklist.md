# 各环节质量检查清单

## 运营分析师（operator）产出检查
- [ ] analytics/report-{account}-{date}.md 存在且包含完整分析
- [ ] 报告包含概览、表现排行、趋势洞察、行动建议
- [ ] 选题建议明确标注为"建议"而非"决策"
- [ ] 洞察输出按角色分类，每条洞察有数据支撑

## 调研员（researcher）产出检查
- [ ] sources.json 包含至少 3 个素材
- [ ] 每个素材有标题、摘要、文件路径
- [ ] outline.md 包含 3-5 个论点
- [ ] 每个论点有案例和出处

## 编剧（writer）产出检查
- [ ] script.md 字数在 1500-2500 字
- [ ] 每个段落有 `> 场景提示：`
- [ ] 每个案例有【出处：...】标注
- [ ] 开头有强钩子（反常识/故事/问题）
- [ ] 语言口语化，适合朗读

## 分镜师（storyboarder）产出检查
- [ ] storyboard.md 存在
- [ ] 镜头编号从 001 开始且连续
- [ ] 每个镜头 2-10 秒
- [ ] 搜索关键词全部为英文
- [ ] 逐字稿每句话都有对应镜头
- [ ] 素材类型分布合理（搜索图片、后期制作、屏幕录制比例适当）
- [ ] 每个镜头有情绪/节奏标注

## 配音师（voice）产出检查
- [ ] audio/voiceover.mp3 文件存在且大小合理
- [ ] audio/subtitles.srt 文件存在
- [ ] 字幕时长与音频时长匹配
- [ ] 字幕内容与原文对应
- [ ] voice-direction.md 存在（如有配音指导需求）

## 美术师（visual）产出检查
- [ ] visuals/ 目录包含所有需要的图片
- [ ] visual-timeline.json 格式正确
- [ ] 每张图片分辨率 ≥ 1920x1080
- [ ] 图片文件完整可打开
- [ ] visual-report.md 存在，记录素材来源和处理情况
- [ ] 后期制作镜头已正确标记（不需要下载素材）

## 剪辑师（editor）产出检查
- [ ] 达芬奇项目文件已创建
- [ ] editor-report.md 存在，记录轨道结构和素材映射
- [ ] V1 轨道：视觉素材按时间线排列
- [ ] V2 轨道：叠加素材/文字/图表（如有）
- [ ] V3 轨道：转场/特效层（如有）
- [ ] A1 轨道：语音旁白
- [ ] A2 轨道：BGM 预留
- [ ] 字幕轨：SRT 字幕已导入
- [ ] 素材路径使用绝对路径
- [ ] 可在 DaVinci Resolve 中正常打开

## 发布员（publisher）产出检查
- [ ] publish/metadata.json 包含标题、描述、标签
- [ ] 各平台标题长度符合限制
- [ ] publish/sources.md 引用出处完整
- [ ] 标签数量和长度符合各平台要求
