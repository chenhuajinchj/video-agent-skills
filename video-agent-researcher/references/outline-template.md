# 大纲模板

## 标准格式

```markdown
# 视频大纲：[主题]

## 核心主题
[一句话概括]

## 目标观众
[描述目标观众特征]

## 预估信息
- 预估时长：X 分钟
- 预估字数：XXXX 字
- 论点数量：X 个

## 开头模式建议
- 推荐模式：反常识/故事/问题
- 理由：[为什么推荐这种模式]

## 论点结构

### 论点 1：[标题]
- 核心观点：[观点]
- 案例：[案例描述]
- 出处：[来源]

### 论点 2：[标题]
- 核心观点：[观点]
- 案例：[案例描述]
- 出处：[来源]

### 论点 3：[标题]
- 核心观点：[观点]
- 案例：[案例描述]
- 出处：[来源]

## 素材摘要

### src_001: [标题]
- 类型：article/youtube
- 内容摘要：[摘要]
```

## 论点组织原则

### 递进式结构（推荐）
1. 现象/问题 → 2. 原因分析 → 3. 解决方案 → 4. 深层启示

### 并列式结构
1. 角度 A → 2. 角度 B → 3. 角度 C → 4. 总结

### 故事式结构
1. 引入故事 → 2. 展开分析 → 3. 反转/深入 → 4. 升华
