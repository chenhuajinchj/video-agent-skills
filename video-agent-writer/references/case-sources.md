# 案例引用指南

本文档说明如何在逐字稿中正确引用案例和标注出处。

## 出处标注格式

### 标准格式

```
【出处：来源类型，具体信息】
```

### 各类来源示例

| 来源类型 | 标注格式 |
|----------|----------|
| 书籍 | 【出处：《书名》第 X 章】 |
| 新闻 | 【出处：媒体名，YYYY 年 M 月 D 日】 |
| 演讲 | 【出处：人名，活动名称，YYYY 年】 |
| 论文 | 【出处：作者，《论文名》，YYYY 年】 |
| 采访 | 【出处：媒体名采访，YYYY 年】 |
| 年报 | 【出处：公司名年度报告，YYYY 年】 |

## 案例类型

### 1. 名人案例

**要求**：
- 必须有明确出处
- 优先使用一手资料（本人著作、演讲、采访）
- 避免二手转述

**示例**：
```
2008 年 9 月，全球金融市场崩溃。
就在所有人疯狂抛售的时候，巴菲特却在《纽约时报》发表文章，
标题是：「Buy American. I Am.」
【出处：纽约时报，2008 年 10 月 17 日】
```

### 2. 数据案例

**要求**：
- 标注数据来源
- 注明统计时间
- 使用权威机构数据

**示例**：
```
根据麦肯锡的研究，
全球 Top 1% 的人掌握了 45% 的财富。
【出处：麦肯锡全球财富报告，2023 年】
```

### 3. 研究案例

**要求**：
- 标注研究者/机构
- 简述研究方法
- 说明关键发现

**示例**：
```
斯坦福大学做过一个著名的「棉花糖实验」。
研究者给孩子们一颗棉花糖，告诉他们：
如果能等 15 分钟不吃，就能得到两颗。
结果发现，能等待的孩子，长大后的成就普遍更高。
【出处：Walter Mischel，斯坦福棉花糖实验，1972 年】
```

### 4. 历史案例

**要求**：
- 时间、地点、人物要准确
- 标注信息来源
- 避免演绎和夸大

**示例**：
```
1997 年，乔布斯重返苹果。
当时苹果濒临破产，股价跌到历史最低。
他做的第一件事，是砍掉 70% 的产品线。
【出处：《乔布斯传》第 28 章】
```

## 常见错误

### 错误 1：无出处

❌ 「有研究表明...」
❌ 「据说...」
❌ 「有人说过...」

✅ 「哈佛商学院的研究表明...【出处：...】」

### 错误 2：出处模糊

❌ 【出处：网上】
❌ 【出处：某本书】
❌ 【出处：巴菲特说的】

✅ 【出处：《滚雪球》第 23 章】

### 错误 3：二手转述

❌ 「我在一篇文章里看到，巴菲特说过...」

✅ 直接引用原始出处

## 案例质量检查清单

- [ ] 案例是否有明确出处？
- [ ] 出处是否为一手资料？
- [ ] 时间、数字是否准确？
- [ ] 案例是否与论点相关？
- [ ] 案例是否足够具体？

## 推荐案例来源

### 书籍类

- 人物传记（《乔布斯传》《滚雪球》等）
- 商业经典（《从优秀到卓越》《原则》等）
- 心理学著作（《思考，快与慢》《影响力》等）

### 媒体类

- 财经媒体（纽约时报、华尔街日报、经济学人）
- 商业杂志（哈佛商业评论、福布斯）
- 官方年报（伯克希尔股东信、公司财报）

### 学术类

- 顶级期刊论文
- 大学研究报告
- 行业白皮书
