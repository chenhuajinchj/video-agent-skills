# 逐字稿风格模板

本文档定义「认知成长/个人成长」类视频的逐字稿写作风格。

## 开头模式

### 模式 1：反常识开头

打破常规认知，制造认知冲突。

**示例**：
```
[画面#001：巴菲特年轻时在办公室看报纸的老照片]
1956 年，26 岁的巴菲特做了一个让所有人不解的决定。

[画面#002：纽约华尔街街景，车水马龙]
当时华尔街正处于黄金年代，所有人都想挤进曼哈顿的投行。

[画面#003：奥马哈小镇的安静街道]
但巴菲特选择回到家乡奥马哈，一个人口不到 30 万的小城市。
他说：「我在这里能更好地思考。」
```

**适用场景**：名人故事、历史事件、违背直觉的观点

### 模式 2：故事开头

从具体场景切入，引发共鸣。

**示例**：
```
[画面#001：深夜办公室，一个人对着电脑]
凌晨两点，小李还在加班。这已经是他这个月第 15 次加班了。

[画面#002：手机屏幕显示银行余额]
他打开手机看了看余额，叹了口气。
工作三年，存款还不到五万。

[画面#003：窗外城市夜景]
他开始怀疑：是不是自己的方向错了？
```

**适用场景**：职场、生活、情感类话题

### 模式 3：问题开头

抛出引人思考的问题。

**示例**：
```
[画面#001：问号动画或思考的人]
为什么有些人工作十年还在原地踏步，
而有些人三年就能实现财务自由？

[画面#002：两条分叉的路]
差距到底在哪里？

[画面#003：书籍或学习场景]
今天我们来聊聊，那些真正改变命运的人，
都做对了什么。
```

**适用场景**：方法论、思维模式、自我提升

## 正文结构

### 论点展开模板

```
[画面#XXX：与论点相关的画面]
【论点陈述】一句话说明核心观点

[画面#XXX：案例相关画面]
【案例展开】
- 具体时间、地点、人物
- 发生了什么
- 结果如何
【出处：来源信息】

[画面#XXX：总结或过渡画面]
【论点总结】这个案例说明了什么
```

### 过渡句式

- 「说到这里，你可能会问...」
- 「但这还不是最关键的...」
- 「接下来这个案例更有意思...」
- 「很多人忽略了一点...」

```

[画面#XXX：关注引导画面]

```

## 语言风格

### 要做的

- 用「你」而不是「您」
- 用短句，每句不超过 20 字
- 用具体数字代替模糊描述
- 用口语化表达，像聊天一样

### 不要做的

- 不要用书面语（「因此」「故而」「综上所述」）
- 不要用空洞的形容词（「非常」「特别」「极其」）
- 不要说教（「你应该」「你必须」）
- 不要堆砌概念

## 画面描述技巧

### 好的描述

- 「巴菲特年轻时在办公室看报纸的老照片」
- 「2008 年金融危机新闻画面，股市暴跌」
- 「手机屏幕显示银行余额」

### 差的描述

- 「一张照片」（太模糊）
- 「金融相关画面」（不具体）
- 「表现焦虑的画面」（太抽象）

### 描述原则

1. 具体：能搜索到或能生成
2. 简洁：10-20 字为宜
3. 可视化：描述看得见的东西
