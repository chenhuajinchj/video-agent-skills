# 认知成长模板（cognitive）

适用主题：个人成长、思维方式、心理学、习惯养成、名人传记解读、方法论分享

## 开头模式表

| 主题类型 | 推荐模式 | 示例 |
|----------|----------|------|
| 名人/历史 | 反常识 | "所有人都以为巴菲特成功靠选股，但他自己说……" |
| 职场/生活痛点 | 故事 | "上周我遇到一个朋友，他每天工作 12 小时但是……" |
| 方法论/思维 | 问题 | "为什么你学了那么多方法，还是觉得……" |

## 结构模板

单核心观点 + 多案例支撑：

```
开头（钩子 + 抛出核心观点）
  │
  ├── 案例 1：从 XX 角度支撑核心观点
  ├── 案例 2：从 YY 角度支撑核心观点
  ├── 案例 3：从 ZZ 角度支撑核心观点
  └── (可选) 案例 4：从反面或延伸角度补充
  │
结尾（回扣核心观点 + 行动号召）
```

整个视频只围绕一个核心观点展开，用多个案例从不同角度支撑。

## 案例组合

推荐的角度组合方式：
- 理论解释 + 科学研究 + 名人故事 + 普通人案例
- 正面案例 + 反面案例 + 数据佐证
- 历史案例 + 当代案例 + 个人体验

每个案例按以下结构展开：
1. **引入**：一句话衔接上一个案例，引出这个案例的角度
2. **故事/数据**：具体的人、事、数据（时间、地点、人物、经过、结果）
3. **出处标注**：独立一行
4. **回扣**：一句话把案例拉回核心观点

## 过渡句示例

- "说到这里，你可能会问……"
- "但这还不是最关键的……"
- "接下来这个案例更有意思……"
- "你可能觉得这跟你没关系，但是……"

## 结尾模式

结尾三要素：
1. 回扣开头的问题/故事/反常识
2. 一句金句收束
3. 一个具体的行动号召（观众看完马上能做的一件事）
